# Cloud vs On prem

## On Premises
- Basically where all the things such as servers, storages and all those things are there in your office itself 
- Full control over the data 
- Security
- Expensive to set up
- Scaling is hard and expensive
- Runs without internet
- Setup can take time
- Maintence should be should

## Cloud
- Uses server like AWS, Azure 
- Can access from anywhere
- Requires strong internet
- Can be setup within minutes
- Maintance will be taken care by the providers

## When to use each
- **CLOUD**: When we want speed and flexibitly
- **ON-Prem**: Use it for high security datas, no reliable internet
