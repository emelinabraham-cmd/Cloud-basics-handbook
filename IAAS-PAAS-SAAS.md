# IAAS VS PAAS VS SAAS
## IAAS(Infrastructure As A Service)
- This is basically an empty plot we hvae to build everything
- We will be given with the cement and all other stuffs
- In Real time it means, we will only be given with the server and we have to install the os, applications and all the other stuff we want

## PAAS(Platform As A Service)
- Here we are given with a house and we have to furnish it and all the stuff
- In real time we just have to code and all things world be ketp ready by the providers

## SAAS(Software As A Service)
- Here we are given with a fully furnished house, we just have to live in it
- In real time it means we only have to use it, just login in and use it
